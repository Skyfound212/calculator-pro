# 🧮 CalculatorPro

Kalkulator profesional dengan mode Basic dan Scientific. Desain modern, cepat, dan akurat untuk kebutuhan kalkulasi harian Anda.

[![Deploy Status](https://img.shields.io/badge/deployed-cloudflare%20pages-orange)](https://calculatorpro.pages.dev)
[![React](https://img.shields.io/badge/react-18.3.1-blue)](https://react.dev)
[![Vite](https://img.shields.io/badge/vite-5.3.1-646CFF)](https://vitejs.dev)

---

## ✨ Fitur

- **Mode Basic** — Aritmatika standar (+, −, ×, ÷), persentase, akar kuadrat
- **Mode Scientific** — Trigonometri (sin, cos, tan), logaritma, pangkat, faktorial, konstanta (π, e)
- **Memory Functions** — MC, MR, M+, M−, MS
- **Riwayat** — Simpan 50 perhitungan terakhir
- **Keyboard Support** — Input cepat via keyboard
- **PWA** — Bisa di-install ke home screen, works offline

---

## 🛠️ Tech Stack

| Layer | Teknologi |
|-------|-----------|
| Framework | React 18 + Vite |
| Routing | React Router DOM v6 |
| Styling | CSS-in-JS |
| Storage | IndexedDB + localStorage |
| PWA | Service Worker + Cache API |
| Deploy | Cloudflare Pages |
