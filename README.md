# CalculatorPro

Kalkulator profesional dengan mode Basic dan Scientific. Desain modern, cepat, dan akurat untuk kebutuhan kalkulasi harian Anda.

## Fitur

- **Mode Basic** — Kalkulasi aritmatika standar (+, −, ×, ÷)
- **Mode Scientific** — Fungsi trigonometri, logaritma, pangkat, akar, faktorial
- **Memory Functions** — MC, MR, M+, M−, MS
- **Riwayat** — Simpan 50 perhitungan terakhir
- **Keyboard Support** — Gunakan keyboard untuk input cepat
- **PWA** — Bisa di-install ke home screen, works offline

## Tech Stack

- React 18
- Vite
- React Router DOM
- CSS-in-JS (no external CSS framework)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
